Отчёт
---
- Приложение реализовано на Java 17 с использованием Spring Boot, Spring MVC, Hibernate, PostgreSQL, Thymeleaf, Gradle, TestNG и Selenium WebDriver. 
- Локальная сборка и запуск выполнялись в окружении Gradle 8.5 и JVM 21.
- Для приложения реализовано тестирование с помощью Selenium. Тесты покрывают как успешные, так и ошибочные сценарии. Покрытие подсчитано инструментом JaCoCo.

Описание страниц
---
На любой странице отображается плашка сверху. Через неё можно перейти на главную страницу, страницу со списком людей, страницу со списком мест, страницу по созданию генеалогического дерева. Для того, чтобы не загромождать диаграмму, данный факт на ней опущен.

![alt text](https://github.com/biscuit-cookie/java-prac-cmc-msu/blob/d14350def1190d94b42746bcfaf1f83ef0db1131/pictures/Java%20task%20webpage%20diagram.drawio.png "Диаграмма страниц")
---
### Главная страница
- Форма для поиска человека по имени. Ссылается на страницу с информацией о человеке
- Ссылка на страницу со списком всех людей
- Ссылка на страницу со списком всех мест
- Ссылка на страницу добавления человека в базу
- Ссылка на страницу для создания дерева

### Страница со списком всех людей
- Список людей, зарегистрированных в базе данных
- На каждого человека можно нажать, тогда мы перейдём на страницу с информацией о нём
- Сортировка по имени человека / его дню рождения - выбрать режим сортировки и нажать "Применить"
- Можно искать людей по имени, не чувствительно к регистру. Для применения фильтра нажать "Применить"
- Кнопка "Добавить человека" - отсылает на соответствующую страницу

### Страница со списком всех мест
- Список мест, зарегистрированных в базе данных
- На каждое место можно нажать, тогда мы перейдём на страницу с информацией о нём
- Можно искать места по имени, не чувствительно к регистру. Для применения фильтра нажать "Применить"
- Кнопка "Добавить место" - отсылает на соответствующую страницу

### Страница с информацией о человеке
- Имя, дата рождения и смерти (смерти - если актуально)
- Краткая характеристика человека
- Выводятся связанные с ним места
- Выводятся связи данного человека с другими людьми. Например, наличие родителей, супруга, детей (если есть информация о них в базе). Указывается начало и конец связи (конец - если указано в базе)
- Можно редактировать информацию о самом человеке - кнопка "Редактировать". Можно построить генеалогическое дерево вокруг данного человека - кнопка "Построить дерево"
- Чтобы добавить информацию о связи с другим человеком, нужно нажать кнопку "Добавить связь". Тогда мы переходим на страницу с формой добавления связи.
- Можно редактировать информацию о связях с другими людьми и можно удалять эти связи. Для этого есть соответствующие кнопки. Редактирование переводит на форму редактирования связи

### Страница добавления человека
- Формы для ввода всех данных о человеке
- Кнопка "Сохранить" для добавления человека с приведённой пользователем информацией, кнопка "Отмена" для возврата на предыдущую страницу.

### Страница добавления места
- Формы для ввода названия места и его описания
- Кнопка "Сохранить" для добавления места с приведённой пользователем информацией, кнопка "Отмена" для возврата на предыдущую страницу.

### Страница для создания дерева
- Кнопка выбора 1 человека/2 человек для построения дерева. 
- Наличие 2-го человека означает что в дереве будет построена отдельная цепочка связи между 1-м человеком и 2-м.
- Выбор параметров дерева. Есть разные способы построения родственных связей: только предки, только потомки, и то и то (смешанный). 
- Можно выбрать желаемую глубину d дерева - оно строится рекурсивно. Отсюда видно родственников до d-го колена, где d - глубина.

### Страница с готовым деревом
- Показывается дерево, созданное по человеку (2 людям) с выбранными параметрами на странице генерации
- Можно скачать дерево в формате JSON - для этого есть кнопка снизу

Сценарии использования
---

- Поиск человека по имени
	- Перейти на главную страницу
	- Перейти на страницу со списком людей
		- Использовать поиск по имени и найти нужного человека
		- Или использовать сортировку по имени или дате рождения
		- Перейти на страницу с информацией о человеке
	* _Или нажать на кнопку "Применить", заранее введя в поле имя_
		- _Перейти на страницу с информацией о человеке_

- Добавление человека в список
    - Перейти на главную страницу
    - Нажать на кнопку "Добавить человека в список"
    - Заполнить всю информацию о человеке
    - Нажать на кнопку "Сохранить" 
    
- Генерация генеалогического дерева человека
	- Перейти на главную страницу
	- Перейти на страницу для генерации дерева
	- Выбрать 1-2 человек для дерева
	* _Или пойти по сценарию поиска человека по имени и на странице с его информацией нажать на кнопку "Создать дерево с человеком"_
	- Настроить параметры дерева
	- Нажать кнопку "Сгенерировать дерево"
	- Перейти на страницу с созданным деревом
	- Скачать дерево
